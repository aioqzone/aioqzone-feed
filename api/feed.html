async get_feeds_by_second(seconds, start=None, *, exceed_pred=None)

Get feeds by abstime (seconds). Range: [start - second, start].

Parameters

  • seconds (float) – filter on abstime, calculate from start.

  • start (Optional[float]) – start timestamp, defaults to None, means now.

  • exceed_pred (Optional[Callable[[FeedRep], Awaitable[bool]]]) – another criterion to judge if the feed is out of range, defaults to None

Raises

  • qqqr.exception.UserBreak – qr login canceled

  • aioqzone.exception.LoginError – not logined

  • SystemExit – unexcpected error

Return type

int

Returns

feeds num got actually.

new_batch()

The new_batch function edit internal batch id and return it.

A batch id can be used to identify a batch, thus even the same feed can have different id e.g. (bid, uin, abstime).

Return type

TY_BID

Returns

The batch_id.
