aioqzone-feed Events

aioqzone-feed follows the hook system defined in aioqzone.interface.hook.

class aioqzone_feed.interface.hook.FeedEvent
async FeedDropped(bid, feed)

The FeedDropped hook is called when a feed is dropped for hitting some rules (e.g. advertisement)

Parameters

  • bid (int) – Used to identify feed batch (tell from different calling).

  • feed (Union[FeedRep, FeedContent]) – Used to pass a ref to the feed.

async FeedMediaUpdate(bid, feed)

The FeedMediaUpdate function is used to update the media of a feed. The aioqzone.api.DummyQapi.floatview_photo_list() is one of the slow api. The media will be update by raw photos/videos, list order should not be changed

Parameters

  • bid (int) – Used to identify feed batch (tell from different calling).

  • feed (FeedContent) – To tell which feed is updated. The feed itself is updated by ref already, the ref should have been passed by FeedProcEnd() in the past.

async FeedProcEnd(bid, feed)

The FeedProcEnd function is called when all processes must be done have finished (i.e. except for slow-api that cannot return at once, and may not matters a lot)

Parameters

  • bid (int) – Used to identify feed batch (tell from different calling).

  • feed (FeedContent) – Used to pass the feed content.

async HeartbeatFailed(exc=None)

The HeartbeatFailed function is called when the heartbeat fails. It can be used to log an error and call aioqzone_feed.api.feed.FeedApi.add_heartbeat() again if possible.

Parameters

exc (Optional[BaseException]) – Used to pass an exception object that can be used to determine the cause of the heartbeat failure.

async HeartbeatRefresh(num)

This event is triggered after a heartbeat succeeded and there are new feeds. Use this event to wait for all dispatch task to be finished, and send received feeds.

Parameters

num (int) – number of new feeds

Example:

async def HeartbeatRefresh(self, num: int):
    await api.get_feeds_by_count(num)
    await api.wait()        # wait for all dispatch tasks and hook tasks
    await queue.send_all()  # send received feeds
